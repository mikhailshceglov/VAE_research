Условие лабораторной работы
1. Обучить вариационный автоэнкодер на CIFAR-10 (запрещено пользоваться готовыми автоэнкодерами. Можно использовать любую нейросеть для энкодера, для декодера, любой самплер, библиотечную DKL, библиотечные функции обучения)
2. Построить “средний” объект в латентном пространстве по каждому классу 
3. Добавив небольшой случайный шум к “среднему” объекту, сгенерировать по 5000 новых объектов каждого класса
4. Обучить на этих новых объектах сеть из первого семестра, валидацию проводить на истинных объектах из CIFAR. Сравнить результаты с лр4 прошлого семестра.
